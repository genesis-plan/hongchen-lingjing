# 红尘灵境·技能库 - 小程序发布信息

## 基本信息
- **小程序名称**: 红尘灵境·技能库
- **简称**: 技能库
- **英文名**: Hongchen Skill Library
- **简介**: 传承千年技艺，开启技能人生。琅琛福地汇聚传统与现代技能，让每个人都能学习改变生活的实用技艺。
- **类目**: 教育 - 在线教育
- **标签**: 技能学习, 传统技艺, 在线教育, 文化传承

## 图标素材
- **Logo设计理念**: 结合传统书卷与现代科技感
- **主色调**: 深蓝色 (#16213e) + 霓虹红 (#ff6b6b)

## 功能介绍
1. **技能浏览**: 5大分类（农业、工艺、艺术、生活、文化）
2. **技能搜索**: 支持名称、描述、分类搜索
3. **技能详情**: 详细介绍、学习目标、课程安排
4. **学习入口**: 一键开始技能学习
5. **收藏功能**: 收藏感兴趣的技能
