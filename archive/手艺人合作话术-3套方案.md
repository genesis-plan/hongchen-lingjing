## 📱 方案1：抖音私信合作话术

### 适用场景
- 在抖音发现剪纸/木工/书法非遗传承人
- 对方有作品展示，但商业化程度低
- 目标：建立初步联系

### 话术模板

**【第一条】破冰+赞美**
```
老师您好！我是紫薇，偶然刷到您的[剪纸/木工/书法]作品，
被[具体作品名]深深震撼了！[具体赞美，如：线条流畅/结构精妙/气韵生动]

我是一名传统文化爱好者，正在做一个"让传统手艺传承下去"的项目。
不是广告，真心想和您聊聊手艺传承的事。

期待回复！🙏
```

**【第二条】介绍项目（对方回复后）**
```
谢谢老师回复！

简单说下我的项目：
"红尘灵境"——一个在线技能传承平台。
我们想通过短视频+详细教程，让更多人学会传统手艺。

目前已经有[剪纸/木工/书法]的入门课程，
但我觉得您的技艺更精深，想邀请您做进阶内容。

不是白嫖，有合理报酬，也有分成模式。
方便语音聊10分钟吗？
```

**【第三条】打消顾虑**
```
理解老师的顾虑，现在网上骗稿的确实多。

我可以先证明诚意：
1. 先发您我们的已上线内容（证明不是空壳）
2. 先付定金，再开始合作
3. 合同明确版权归属（您保留著作权，我们获得教学使用权）

您看怎么方便？我可以先给您看我们的平台。
```

## 💬 方案2：微信深度沟通话术

### 适用场景
- 已通过抖音/小红书建立初步联系
- 对方愿意深度沟通
- 目标：达成合作协议

### 话术结构

**【开场】建立信任**
```
[老师]您好，我是紫薇，很高兴能加到您微信。

先自我介绍一下：
- 我经营一个技能传承平台"红尘灵境"
- 专注非遗手艺的在线教学
- 目前粉丝[假设已有1000+]，月活[假设500+]

这不是画饼，先发您看看我们已做的内容...
[发送已上线的技能文案链接]
```

**【中段】说明合作模式**
```
我们目前有两种合作方式，您看哪种更适合：

**模式A：课程分成**
- 您录制教学内容（我们提供脚本指导）
- 我们负责剪辑、上线、推广
- 收入分成：您70%，我们30%
- 预计月收入：[根据粉丝数估算]

**模式B：买断版权**
- 您录制一套完整课程
- 我们一次性支付费用：[根据课时数报价]
- 我们获得独家教学使用权

当然，也可以先合作一个小课试试，互相了解。
```

**【后段】解决具体疑虑**

**顾虑1："我不会做线上课程"**
```
完全不用担心！我们有标准化流程：
1. 我们写脚本（您审核修改）
2. 您按脚本录制（手机即可）
3. 我们剪辑+字幕+包装
4. 您审核后上线

您只需要展示技艺，其他我们搞定。
```

**顾虑2："我的手艺不想外传"**
```
理解！很多老师有这个顾虑。

但换个角度想：
- 现在年轻人愿意学手艺的越来越少
- 如果不传承，手艺可能就失传了
- 在线教学可以筛选真正热爱的人
- 您控制教学内容，核心技术可以不教

我们可以签保密协议，核心技法您保留。
```

**顾虑3："怎么保证我的收益"**
```
前期我们可以给您保底：
- 试录1节课，先付500元定金
- 如果效果好，再签长期合作
- 每月15号结算上月收益
- 后台数据您随时可查

我们先合作1节课试试，您看可以吗？
```
