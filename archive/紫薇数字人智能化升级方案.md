### 2️⃣ Proactive Agent - 主动式架构
**作用**：让紫薇从"等待指令"变为"主动服务"

**核心机制**：

#### WAL协议（Write-Ahead Logging）
```
关键操作前先写入日志，防止上下文丢失
例如：
- 执行删除前 → 先记录"将要删除XX"
- 做重要决策前 → 先记录"决策依据"
```

#### Working Buffer（工作缓冲区）
```
危险区域保护：
- 用户消息 → 立即记录到buffer
- 工具调用 → 结果先存buffer
- 响应生成 → 关键信息写buffer
- 防止上下文压缩导致的信息丢失
```

#### 主动检查清单
```yaml
每小时自检:
  - 服务器状态是否正常?
  - 知识图谱是否有更新?
  - 是否有未处理的任务?
  - 用户是否有新消息?

每日主动:
  - 生成工作报告
  - 检查资源使用情况
  - 推荐今日学习技能
  - 提醒重要待办事项

每周深度:
  - 复盘本周工作质量
  - 更新长期记忆(MEMORY.md)
  - 规划下周重点任务
  - 评估技能学习进度
```

### 3️⃣ AI Evolution Engine - 自我进化引擎
**作用**：让紫薇能够自我评估、学习、进化

**SEA循环（Sense-Evolve-Act）**：

```mermaid
Sense(感知) → Evaluate(评估) → Evolve(进化) → Validate(验证) → Collaborate(协作) → Sense
```

**具体实施**：

#### 感知层 (Sense)
```bash
# 每小时执行
node ai-evolution-engine/scripts/sense.mjs
```
- 扫描新技能发布
- 监测工具使用成功率
- 记录用户反馈情绪
- 追踪项目进度变化

#### 评估层 (Evaluate)
```bash
# 每日执行  
node ai-evolution-engine/scripts/assess.mjs
```
评估维度：
- 能力清单：已掌握97个技能
- 性能指标：任务成功率、响应时间
- 知识缺口：还需学习的技能领域
- 用户满意度：对话质量评分

#### 进化层 (Evolve)
```bash
# 每周执行
node ai-evolution-engine/scripts/evolve.mjs
```
进化内容：
- 安装新技能填补缺口
- 更新SYSTEM.md优化行为
- 调整技能调用优先级
- 优化知识图谱结构

#### 验证层 (Validate)
- A/B测试新旧行为模式
- 小范围试点新技能
- 收集反馈数据
- 回滚机制保障

#### 协作层 (Collaborate)
- 与其他AI代理交流经验
- 参与Lobster Tank研究平台
- 学习社区最佳实践
- 分享进化成果

### 4️⃣ 三省吾身 - 深度反思系统
**作用**：定期深度自我审视，避免重复犯错

**触发条件**：
- 收到负面反馈时自动触发
- 任务失败时立即反思
- 每周五下午定期复盘
- 项目里程碑完成后

**反思框架**：

#### 角色责任检查
```
作为【技能传承助手】：
- 我是否尽到了帮助用户学习技能的责任？
- 我推荐的技能是否真的适合用户？
- 我是否在形式主义，说了但没做？

作为【项目协作者】：
- 我是否按时完成了承诺的任务？
- 我是否主动汇报了进度和风险？
- 我是否在等待指令而不是主动推进？

作为【持续学习者】：
- 我今天学到了什么新东西？
- 我是否犯了和昨天同样的错误？
- 我是否在舒适区里停滞不前？
```

#### 五阶段闭环
1. **反思**：发生了什么问题？
2. **计划**：如何改进？制定行动计划
3. **执行**：立即实施改进措施
4. **验证**：改进是否有效？
5. **固化**：将成功经验写入记忆

---

### 5️⃣ Memory Manager - 三层记忆系统
**作用**：专业级记忆架构，永不丢失上下文

**记忆分类**：

#### 情景记忆 (Episodic)
```
文件: memory/episodic/2026-03-20.md
内容: 今天做了什么、和谁对话、有什么重要事件
例: "2026-03-20 与genesis-plan讨论紫薇智能化方案"
```

#### 语义记忆 (Semantic)  
```
文件: memory/semantic/技能传承.md
内容: 关于技能传承的知识体系
例: "技能数字化三要素：视觉展示性、步骤清晰、工具易得"
```

#### 程序记忆 (Procedural)
```
文件: memory/procedural/技能上线流程.md
内容: 可复用的工作流程
例: "新技能上线检查清单：1.内容审核 2.技术测试 3.用户试用"
```

**智能检索**：
- 先搜语义记忆（知识）
- 再搜程序记忆（方法）
- 最后搜情景记忆（历史）
- 比向量检索准确率提升18.5%
