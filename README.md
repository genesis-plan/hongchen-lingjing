## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + JavaScript + Three.js
- **后端**: 腾讯云 CloudBase (云函数 + 数据库)
- **部署**: CloudBase 静态托管 / GitHub Pages

---

## 📖 快速开始

```bash
# 克隆项目
git clone https://github.com/genesis-plan/hongchen-lingjing.git

# 进入网页版目录
cd hongchen-lingjing/web

# 启动本地服务器
python -m http.server 8080
# 或
npx http-server

# 访问 http://localhost:8080
```
