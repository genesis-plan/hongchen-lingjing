## Genesis Plan 创世计划核心项目 · 元境大同

> 赛博国风全球技能传承开源元宇宙 | An open-source metaverse for cyberpunk-Chinese skill inheritance & co-creation
> 官网：[红尘灵境.com](https://hongchenlingjing.com)

### 🌟 创世缘起 | Origin
当代码成为刻刀，当数字空间成为故土，我们想做一件事——让中国的榫卯、剪纸、中医推拿，让全球的皮具制作、法式烘焙、开源编程，都能在赛博国风的元宇宙里被看见、被学习、被传承。
Genesis Plan 创世计划，以开源为骨，以国风为魂，邀全球同频者，共筑元境大同，让技能跨越国界，生生不息。

### 🌌 项目愿景 | Vision
1. **技能无界**：汇聚全球传统/现代技能，打造标准化数字传承体系；
2. **国风为核**：以赛博国风为视觉与体验内核，传递中国文化精神；
3. **开源共创**：基于MIT协议开放所有内容，全球开发者共建共治；
4. **扎根中国**：核心决策归属中国治理会，坚守技能传承初心。

### 🛠 快速开始 | Quick Start
无需开发环境，直接运行：
1. 克隆仓库：`git clone https://github.com/genesis-plan/hongchen-lingjing.git`
