## 📊 项目概览 | Project Overview

### 🏗️ 项目架构 | Project Architecture

红尘灵境采用**多端协同**架构，包括网页版和微信小程序：

```
红尘灵境
├── 🌐 网页版 (Web)
│   ├─ 技术栈: HTML5 + CSS3 + JavaScript + Three.js
│   ├─ 部署: 腾讯云 CloudBase 静态托管
│   └─ 访问: https://红尘灵境.com
│
└── 📱 微信小程序 (Mini Program)
    ├─ 技术栈: 原生小程序框架
    ├─ 后端: 腾讯云 CloudBase 云函数
    └─ 数据库: 15个集合（用户、技能、愿望等）
```

### 📦 核心模块 | Core Modules

1. **🌾 职业系统**：农民作为首个创始职业
2. **📚 技能系统**：学习、实践、认证全流程
3. **🤖 AI对话**：智能助手辅助学习
4. **🏆 荣誉体系**：技能证书、成就勋章
5. **💬 社交系统**：好友、书信、社区
6. **💰 经济系统**：积分交易、技能市场
