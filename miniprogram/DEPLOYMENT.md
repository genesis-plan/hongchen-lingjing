## 详细部署步骤

### 第一步：创建小程序项目

1. 打开微信开发者工具
2. 点击"+"号，新建项目
3. 填写项目信息：
   - 项目名称：红尘灵境
   - 目录：选择项目文件夹
   - AppID：填写你的小程序 AppID
   - 开发模式：小程序
   - 后端服务：微信云开发
4. 点击"新建"

### 第二步：配置云开发

1. 开发者工具中点击"云开发"按钮
2. 根据提示开通云开发
3. 创建环境：
   - 环境名称：hclj-production
   - 环境ID：hclj-8g46g9fd06e2a760
   - 基础版配置（免费额度）

### 第三步：配置项目

1. 打开 `project.config.json`
2. 修改 AppID 为你的 AppID
3. 打开 `app.js`
4. 确认环境ID正确：`env: 'hclj-8g46g9fd06e2a760'`

### 第四步：初始化数据库

1. 进入云开发控制台
2. 创建以下集合：
   - `users` - 用户表
   - `skills` - 技能表
   - `wishes` - 心愿表
   - `honors` - 荣誉表
   - `examResults` - 考核结果表

3. 导入初始数据：
   - 在云开发控制台数据库页面
   - 选择 `skills` 集合
   - 点击"导入"
   - 选择 `database-init.js` 中的技能数据
   - 格式选择：JSON

### 第五步：上传云函数

1. 在开发者工具中，右键 `cloudfunctions` 文件夹
2. 选择"上传并部署：云端安装依赖"

3. 依次上传所有云函数：
   - `login`
   - `getSkills`
   - `submitWish`
   - `saveProgress`
   - `saveExamResult`
   - `saveCharacter`
   - `getUserStats`

### 第六步：测试小程序

1. 点击"编译"按钮
2. 在模拟器中测试功能：
   - [ ] 用户登录
   - [ ] 技能浏览
   - [ ] 技能学习
   - [ ] 技能考核
   - [ ] 荣誉查看
   - [ ] 个人空间

3. 使用真机预览功能，在手机上测试

### 第七步：上传小程序

1. 点击工具栏的"上传"按钮
2. 填写版本号：1.0.0
3. 填写项目备注：首个测试版本
4. 点击"上传"

### 第八步：提交审核

1. 登录微信公众平台
2. 进入"版本管理"
3. 找到刚上传的版本
4. 点击"提交审核"
5. 填写审核信息：
   - 功能页面：灵境、琅嬛、实训、荣誉、空间
   - 类目选择：教育 > 培训
6. 等待审核（通常1-7天）

### 第九步：发布上线

1. 审核通过后
2. 在版本管理中点击"发布"
3. 确认发布
4. 小程序正式上线

## 数据库权限配置

在云开发控制台的"数据库" -> "权限设置"中：

```json
{
  "read": true,
  "write": false
}
```

所有数据操作通过云函数进行，保证安全性。

## 云函数权限配置

每个云函数需要添加环境权限：
- 读取数据库权限
- 写入数据库权限

## 常见问题解决

### 问题1：云函数调用失败
**解决方案：**
- 检查云开发环境ID是否正确
- 确认云函数已上传部署
- 查看云函数日志定位错误

### 问题2：数据无法保存
**解决方案：**
- 检查数据库权限设置
- 确认云函数有读写权限
- 查看网络请求是否正常

### 问题3：页面加载慢
**解决方案：**
- 开启云函数缓存
- 优化数据库查询
- 使用CDN加速图片资源

### 问题4：审核不通过
**解决方案：**
- 检查小程序类目是否正确
- 确保没有违规内容
- 添加必要的隐私政策页面

## 性能优化建议

1. **图片优化**
   - 使用 WebP 格式
   - 开启图片懒加载
   - 使用 CDN 加速

2. **代码优化**
   - 压缩 JS 代码
   - 减少 HTTP 请求
   - 使用分包加载

3. **缓存策略**
   - 合理使用缓存
   - 设置适当的缓存时间
   - 定期清理过期缓存
