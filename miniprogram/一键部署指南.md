## 快速部署步骤（5步完成）

### 第一步：安装微信开发者工具

1. 访问官网下载：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 选择Windows版本下载
3. 安装并打开微信开发者工具
4. 使用微信扫码登录

### 第二步：导入项目

1. 打开微信开发者工具
2. 点击 "+" 号新建项目
3. 填写以下信息：
   - **项目名称**：红尘灵境
   - **目录**：选择 `C:\Users\Administrator\Desktop\红尘灵境小程序`
   - **AppID**：选择"测试号"（如果没有正式AppID）
   - **开发模式**：小程序
   - **后端服务**：微信云开发
4. 点击"新建"

### 第三步：配置云开发环境

1. 在开发者工具顶部菜单，点击"云开发"按钮
2. 首次使用需要开通云开发，点击"开通"
3. 选择"按量付费"（免费额度足够测试使用）
4. 创建环境：
   - **环境名称**：hclj-production
   - **环境ID**：hclj-8g46g9fd06e2a760（已有环境直接选择）
5. 点击"创建"或"选择"

### 第四步：上传云函数（一次性操作）

1. 在开发者工具左侧文件树中，找到 `cloudfunctions` 文件夹
2. 右键点击每个云函数文件夹，选择"上传并部署：云端安装依赖"
3. 需要上传的云函数（共18个）：
   - `login` - 用户登录
   - `getSkills` - 获取技能列表
   - `submitWish` - 提交心愿
   - `saveProgress` - 保存学习进度
   - `saveExamResult` - 保存考核结果
   - `saveCharacter` - 保存个人形象
   - `getUserStats` - 获取用户统计
   - `createSkill` - 创建技能
   - `askAI` - AI问答
   - `getLetters` - 获取信件
   - `sendLetter` - 发送信件
   - `markLetterRead` - 标记信件已读
   - `getPosts` - 获取帖子
   - `createPost` - 创建帖子
   - `getActivities` - 获取活动
   - `joinActivity` - 参加活动
   - `sendFriendRequest` - 发送好友请求
   - `getFriendRequests` - 获取好友请求
   - `acceptFriendRequest` - 接受好友请求

4. 等待所有云函数上传完成（每个约10-30秒）

### 第五步：创建数据库集合（一次性操作）

1. 在开发者工具顶部菜单，点击"云开发"按钮
2. 左侧菜单选择"数据库"
3. 点击"添加集合"，创建以下19个集合：
   - `users` - 用户信息
   - `skills` - 技能库
   - `wishes` - 心愿池
   - `honors` - 荣誉成就
   - `examResults` - 考核结果
   - `userCharacters` - 个人形象
   - `letters` - 信件
   - `posts` - 社区帖子
   - `activities` - 社区活动
   - `friendRequests` - 好友请求
   - `friends` - 好友关系
   - `marketItems` - 市场物品
   - `transactions` - 交易记录
   - `challenges` - 挑战任务
   - `challengeRecords` - 挑战记录

4. 设置数据库权限：
   - 在数据库页面，点击"权限设置"
   - 选择"所有用户可读，仅创建者可写"

### 第六步：测试运行

1. 在开发者工具顶部，点击"编译"按钮
2. 模拟器会显示小程序界面
3. 测试基本功能：
   - 点击"进入灵境"按钮
   - 查看首页
   - 点击底部导航栏切换页面
   - 查看各殿功能

## 正式发布（需要正式AppID）

### 获取正式AppID

1. 访问 https://mp.weixin.qq.com/
2. 使用微信扫码登录
3. 点击"立即注册"
4. 选择"小程序"
5. 填写注册信息
6. 完成认证（个人或企业）

### 替换AppID

1. 在开发者工具中，打开 `project.config.json`
2. 找到 `"appid"` 字段
3. 将测试号替换为正式AppID
4. 保存文件

### 上传代码

1. 在开发者工具顶部，点击"上传"按钮
2. 填写版本号：1.0.0
3. 填写项目备注：红尘灵境v1.0正式版
4. 点击"上传"

### 提交审核

1. 登录 https://mp.weixin.qq.com/
2. 进入"版本管理"
3. 找到上传的版本
4. 点击"提交审核"
5. 填写审核信息：
   - 功能页面：灵境、琅嬛、实训、考核、荣誉、空间等
   - 类目选择：教育 > 培训
   - 服务类标签：教育培训、学习辅助

### 发布上线

1. 审核通过后（通常1-7天）
2. 在版本管理中点击"发布"
3. 确认发布
4. 小程序正式上线，用户可搜索使用

## 常见问题

### Q1：提示"云函数未部署"
**A**：检查云开发环境ID是否正确，确保所有云函数都已上传成功

### Q2：页面打不开
**A**：确保 `app.json` 中的页面路径正确，检查对应文件是否存在

### Q3：数据无法保存
**A**：检查数据库权限设置，确认云函数已正确部署

### Q4：编译报错
**A**：检查语法错误，查看控制台错误信息，确保所有文件都存在

### Q5：测试号和正式AppID有什么区别？
**A**：
- 测试号：仅用于开发测试，功能有限，无法发布上线
- 正式AppID：可以发布上线，所有功能可用，用户可搜索使用
