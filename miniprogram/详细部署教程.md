# 红尘灵境小程序 - 详细部署教程（图文版）

## 📌 重要提示

我已经为您打开了微信开发者工具的下载页面，请在浏览器中完成下载和安装。

---

## 🔍 第一部分：下载和安装微信开发者工具

### 步骤1：下载微信开发者工具

1. 浏览器已自动打开下载页面
2. 在页面中找到"微信开发者工具"下载区域
3. 点击"Windows"版本的下载按钮
4. 等待下载完成（文件名类似：wechat_devtools_x.x.x.exe）

### 步骤2：安装微信开发者工具

1. 找到下载的安装包（通常在"下载"文件夹）
2. 双击运行安装包
3. 点击"安装"或"下一步"
4. 等待安装完成
5. 安装完成后，双击桌面上的"微信开发者工具"图标打开

### 步骤3：登录微信开发者工具

1. 打开微信开发者工具后，会显示二维码
2. 使用微信扫码登录
3. 登录成功后进入欢迎界面

---

## 🚀 第二部分：导入项目

### 步骤4：导入红尘灵境项目

1. 在微信开发者工具首页，点击左上角的 **"+"号** 按钮
2. 选择 **"导入项目"**
3. 填写以下信息：

   - **项目名称**：红尘灵境
   - **项目目录**：点击"选择"按钮，然后导航到并选择：
     `C:\Users\Administrator\Desktop\红尘灵境小程序`
   - **AppID**：选择 **"测试号"** （如果没有正式AppID）
   - **开发模式**：选择 **"小程序"**
   - **后端服务**：选择 **"微信云开发"**

4. 点击"导入"按钮
5. 等待项目加载完成

### 步骤5：查看项目结构

导入成功后，您会看到：
- 左侧是文件树，显示所有页面和云函数
- 中间是代码编辑区
- 右侧是模拟器，显示小程序界面

## ☁️ 第三部分：配置云开发环境

### 步骤6：开通云开发服务

1. 在微信开发者工具顶部菜单栏，点击 **"云开发"** 按钮
2. 首次使用会提示开通云开发，点击 **"开通"**
3. 选择付费方式：
   - 选择 **"按量付费"** （有免费额度，测试足够）
4. 填写环境信息：
   - **环境名称**：hclj-production
   - **环境ID**：hclj-8g46g9fd06e2a760
5. 点击 **"创建"** 或 **"选择"**

### 步骤7：等待环境创建完成

- 环境创建需要几分钟时间
- 创建完成后会自动进入云开发控制台

## 📱 正式发布（可选）

如果您想让其他人也能使用小程序，需要进行正式发布：

### 步骤17：获取正式AppID

1. 访问：https://mp.weixin.qq.com/
2. 注册微信小程序账号
3. 完成认证
4. 获取正式AppID

### 步骤18：替换AppID

1. 在微信开发者工具中，打开 `project.config.json`
2. 找到 `"appid"` 这一行
3. 把 `"appid": ""` 改成 `"appid": "你的正式AppID"`

### 步骤19：上传代码

1. 点击工具栏的 **"上传"** 按钮
2. 填写版本号：`1.0.0`
3. 填写项目备注：`红尘灵境正式版`
4. 点击"上传"

### 步骤20：提交审核

1. 登录微信后台：https://mp.weixin.qq.com/
2. 进入"版本管理"
3. 找到上传的版本
4. 点击"提交审核"
5. 填写审核信息并提交

### 步骤21：发布上线

1. 审核通过后（通常1-7天）
2. 在版本管理中点击"发布"
3. 确认发布
4. 小程序正式上线，用户可搜索使用

## ❓ 常见问题

### Q1：云函数上传失败怎么办？
**A**：
- 检查网络连接
- 重新右键云函数文件夹
- 再次选择"上传并部署：云端安装依赖"

### Q2：数据库集合创建失败怎么办？
**A**：
- 确保云开发环境已开通
- 检查集合名称是否正确（不要有空格）
- 删除失败的集合，重新创建

### Q3：编译报错怎么办？
**A**：
- 查看工具底部的控制台错误信息
- 检查代码文件是否完整
- 清除缓存重新编译

### Q4：页面打不开怎么办？
**A**：
- 检查 `app.json` 中的页面路径是否正确
- 确认对应的页面文件都存在
- 重新编译项目

### Q5：测试号和正式AppID有什么区别？
**A**：
- **测试号**：仅用于开发测试，无法发布上线
- **正式AppID**：可以发布上线，用户可搜索使用
