## 项目简介

**红尘灵境**是一款融合修仙主题的学习管理系统。通过游戏化的方式，让学习变得有趣且可持续。

## 核心功能

| 功能模块 | 说明 |
|---------|------|
| 🏛️ **十四建筑** | 14个功能建筑，覆盖学习全流程 |
| 📚 **琅嬛藏书** | 100+技能内容库 |
| 🎓 **修炼实训** | 实践导向的学习项目 |
| 🏆 **荣誉殿堂** | 成就激励系统 |
| 🤖 **紫薇AI** | 24小时智能学习助手 |

## 技术架构

- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: 腾讯云云函数 + CloudBase
- **AI**: 自主学习系统 (24小时自动化)
- **部署**: GitHub Pages / COS / 静态托管
