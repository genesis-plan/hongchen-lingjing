# 任务队列使用说明

## 目录结构

```
任务队列/
├── 待分配/              # 新任务放这里
│   └── 任务-完善天工开物坊-20260317-1845.txt
├── 紫薇执行中/          # 紫薇正在执行的任务
├── 龙虾执行中/          # 龙虾正在执行的任务
├── 已完成/              # 已完成的任务
│   └── 任务-xxx-完成-20260317.md
└── 紧急/                # 高优先级任务
    └── 紧急-服务器故障-20260317.md
```

## 任务文件格式

```markdown
# 任务: 任务名称

## 优先级
P0/P1/P2/P3

## 任务描述
详细描述任务内容

## 负责人
紫薇/龙虾助手/任意

## 预期结果
明确的目标

## 验证方式
可执行的命令或检查方法

## 截止时间
2026-03-17 20:00

## 创建时间
2026-03-17 18:45
```

## 使用流程

### 1. 创建任务
在`待分配/`目录创建任务文件

### 2. 认领任务
智能体检查后，将任务移动到自己的执行中目录

### 3. 执行任务
更新任务文件添加执行进度

### 4. 完成任务
移动到`已完成/`目录，更新状态

## 状态同步

- 每小时检查一次`待分配/`
- 认领时在文件名后加`_锁定_{智能体}`
- 执行中更新进度到`任务文件.md`
- 完成后移动到`已完成/`
