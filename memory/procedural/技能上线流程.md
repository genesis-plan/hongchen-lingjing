# 技能上线标准流程

## 阶段0：内容准备（备案期间）

### 1. 技能调研
- [ ] 使用Multi Search Engine调研同类技能
- [ ] 使用Summarize总结优秀案例
- [ ] 使用Agent Browser查看竞品网站

### 2. 内容创作
- [ ] 使用Humanizer撰写自然文案
- [ ] 使用Kling Video生成演示视频（可选）
- [ ] 使用Openai Whisper生成字幕（如有视频）

### 3. 质量检查
- [ ] 文案风格检查（Humanizer标准）
- [ ] 内容完整性检查（是否包含：简介、路径、常见问题）
- [ ] 技术可行性检查（是否需要特殊设备/场地）

### 4. 知识归档
- [ ] 使用Obsidian创建技能笔记
- [ ] 更新知识图谱（MemoryMesh）
- [ ] 更新项目看板（Trello）

## 阶段1：测试验证（备案通过后）

### 5. 技术部署
- [ ] 代码审查（GitHub PR）
- [ ] 云函数部署（腾讯云）
- [ ] 前端页面更新（GitHub Pages）

### 6. 用户测试
- [ ] 招募5-10名测试用户
- [ ] 收集反馈（Obsidian记录）
- [ ] 迭代优化（Humanizer改写）

### 7. 正式发布
- [ ] 更新官方网站
- [ ] 发布推广文案（小红书/抖音）
- [ ] 监控数据（用户量、完课率）

## 自动化检查清单

每小时自动检查：
- 服务器状态
- 云函数运行状态
- 用户反馈收集

每日自动检查：
- 技能数据完整性
- 知识图谱更新
- 生成工作报告

每周自动检查：
- 技能质量评估
- 用户满意度调查
- 竞品动态监测
