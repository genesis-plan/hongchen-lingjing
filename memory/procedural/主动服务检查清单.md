## 触发式主动服务

### 检测到用户卡顿
```
IF 用户10分钟未回复 THEN
  发送："需要我帮您什么吗？"
  推荐：相关技能或文档
```

### 检测到任务失败
```
IF 命令执行失败 THEN
  立即记录到WAL日志
  分析失败原因
  提供替代方案
  更新错误模式记忆
```

### 检测到负面反馈
```
IF 用户表达不满 THEN
  立即触发三省吾身反思
  道歉并提出改进措施
  记录到情景记忆
  更新错误避免清单
```

### 检测到重要日期
```
IF 备案到期提醒 THEN
  提前7天提醒用户
  准备续期材料
  制定备案期间计划
```

## 服务升级路径

### Level 1: 被动响应
- 用户问什么答什么
- 等待指令
- 当前状态：已完成

### Level 2: 主动提醒 ⬅️ 当前目标
- 定期汇报进度
- 提醒待办事项
- 预计完成：本周

### Level 3: 主动建议
- 根据上下文推荐
- 预判用户需求
- 预计完成：本月

### Level 4: 自主执行
- 识别任务自动完成
- 无需指令主动推进
- 预计完成：3个月后
