# 目录说明

本目录包含红尘灵境网页版（Web版）的完整源代码。

## 项目说明

红尘灵境网页版是一个元宇宙技能学习平台，提供：

- 技能学习系统
- 14个虚拟建筑探索
- 学习进度追踪
- 用户互动功能

## 技术栈

- 纯 HTML/CSS/JavaScript
- 腾讯云云开发（静态托管）
- GitHub Pages（备用部署）
