## 📝 使用说明

### 添加任务
```bash
./scripts/task-add.sh "任务描述" [优先级P0/P1/P2/P3] [截止时间]
```

### 完成任务
```bash
./scripts/task-complete.sh 任务编号 [交付物] [验证结果]
```

### 查看任务
```bash
./scripts/task-list.sh [进行中/已完成/待处理/all]
```

### 生成日报
```bash
./scripts/task-daily-report.sh
```
