## 🚀 快速开始 | Quick Start

### 最简单的方式 - 在线体验

**直接访问**: https://红尘灵境.com

无需下载、无需安装，直接在浏览器中体验红尘灵境。

### 本地体验（开发者）

如果您想查看代码或本地运行：

```bash
# 1. 克隆仓库
git clone https://github.com/genesis-plan/hongchen-lingjing.git

# 2. 进入网页目录
cd hongchen-lingjing/web

# 3. 启动本地服务器
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx http-server

# 4. 访问
# 打开浏览器访问 http://localhost:8080
```

## 🌐 网页版使用 | Web Version

### 访问网站

1. **打开浏览器**
   - 推荐使用 Chrome、Firefox、Safari 或 Edge

2. **输入网址**
   - https://红尘灵境.com

3. **开始探索**
   - 首页会展示红尘灵境的世界观介绍
   - 点击导航菜单可以访问不同功能

### 用户注册

1. **点击注册**
   - 在首页点击"注册"按钮

2. **填写信息**
   - 用户名：3-20个字符
   - 邮箱：有效的邮箱地址
   - 密码：至少6个字符
   - 确认年龄：必须年满14周岁

3. **同意条款**
   - 阅读并同意隐私政策
   - 阅读并同意服务条款
   - 点击"注册"按钮

4. **完成注册**
   - 系统会创建您的账户
   - 您可以立即开始使用

### 用户登录

1. **点击登录**
   - 在首页点击"登录"按钮

2. **输入凭证**
   - 用户名或邮箱
   - 密码

3. **登录成功**
   - 进入个人主页
   - 开始您的红尘灵境之旅

### 年龄验证

根据中国法律法规，我们要求所有用户必须年满14周岁：

1. **验证过程**
   - 注册时会要求确认年龄
   - 系统会记录您同意的时间

2. **隐私保护**
   - 您的年龄信息严格保密
   - 仅用于年龄验证

### 技能图书馆

1. **访问图书馆**
   - 点击导航菜单的"技能图书馆"

2. **浏览技能**
   - 滚动浏览所有可用技能
   - 点击技能卡片查看详情

3. **学习技能**
   - 阅读技能说明
   - 了解技能要求
   - 查看学习路径

### 实践场

1. **进入实践场**
   - 点击导航菜单的"实践场"

2. **选择练习**
   - 浏览可用的练习项目
   - 选择适合您水平的练习

3. **开始练习**
   - 按照指导完成练习
   - 记录您的进度

## 📱 小程序使用 | Mini Program

### 安装小程序

1. **打开微信**
   - 确保微信版本 7.0 或以上

2. **搜索小程序**
   - 点击微信右上角的"+"
   - 选择"扫一扫"
   - 或点击"发现" → "小程序"

3. **搜索名称**
   - 输入"红尘灵境"
   - 或扫描提供的二维码

4. **添加小程序**
   - 点击"添加到我的小程序"
   - 方便日后访问

### 用户授权

1. **首次打开**
   - 小程序会请求用户信息授权

2. **授权微信登录**

3. **完善信息**
   - 设置昵称
   - 选择头像
   - 创建您的数字分身

### 首页导航

1. **查看首页**

   - 查看推荐内容

2. **底部导航栏**

### 技能图书馆（小程序）

1. **浏览技能**

2. **查看详情**

3. **开始学习**

### 实践场（小程序）

1. **选择练习**

2. **完成练习**

3. **查看成果**
   - 查看练习结果
   - 查看获得的经验

### 技能市场

1. **浏览市场**
   - 查看可购买的技能
   - 查看技能价格

2. **购买技能**
   - 使用积分购买
   - 立即解锁新技能

3. **交易记录**
   - 查看购买历史
   - 管理已购技能

### 社交功能

1. **添加好友**
   - 搜索用户
   - 发送好友请求
   - 等待对方同意

2. **发送书信**
   - 选择好友
   - 编写书信内容
   - 发送书信

3. **社区互动**
   - 浏览社区帖子
   - 点赞、评论
   - 发布自己的内容

### AI对话

1. **访问AI助手**
   - 点击AI图标
   - 开始对话

2. **提出问题**

## 🎯 核心功能详解 | Core Features

### 🌾 职业系统

#### 农民职业

红尘灵境的首个创始职业，代表文明的起点。

**功能**:
- 🌱 虚拟种植
- 🌾 农作物管理
- 🌾 收成系统
- 🏆 成就解锁

**如何开始**:
1. 注册/登录后
2. 在个人中心选择职业
3. 选择"农民"
4. 开始您的农耕之旅

### 📚 技能系统

#### 技能学习

技能是红尘灵境的核心，通过学习技能可以提升自己的能力。

**学习流程**:
1. 在技能图书馆浏览技能
2. 选择感兴趣或需要的技能
3. 阅读技能说明
4. 开始学习并记录进度

#### 技能认证

完成学习后，可以通过认证获得证书。

**认证流程**:
1. 在实践场完成练习
2. 参加技能考试
3. 通过后获得证书
4. 证书展示在荣誉殿堂

### 🏆 荣誉系统

#### 成就勋章

完成特定任务或达到特定条件会获得成就勋章。

**成就类型**:
- 🥇 学习成就
- 🥈 实践成就
- 🥉 社交成就
- 🌟 特殊成就

#### 技能证书

通过技能认证后获得证书。

**证书展示**:
- 在荣誉殿堂查看
- 在个人资料展示
- 分享到社交平台

### 💬 社交系统

#### 好友系统

与其他用户建立联系。

**好友功能**:
- 添加好友
- 好友列表
- 好友状态
- 互动消息

#### 书信系统

通过书信与好友交流。

**书信功能**:
- 发送书信
- 接收书信
- 书信历史
- 书信标记

### 💰 经济系统

#### 积分系统

通过活动获得积分。

**获得积分**:
- 每日签到
- 完成练习
- 技能认证
- 社交互动

**使用积分**:
- 购买技能
- 解锁特殊功能
- 兑换奖励

## ❓ 常见问题 | FAQ

### 网页版问题

**Q: 网页加载很慢怎么办？**
A:
- 检查您的网络连接
- 清除浏览器缓存
- 尝试使用其他浏览器
- 检查是否使用了VPN

**Q: 3D效果无法显示？**
A:
- 确保您的浏览器支持WebGL
- 更新浏览器到最新版本
- 检查显卡驱动是否最新
- 暂时禁用浏览器扩展

**Q: 忘记密码怎么办？**
A:
- 目前版本不支持自助找回
- 请联系客服重置密码
- 未来版本会增加自助找回功能

### 小程序问题

**Q: 小程序无法打开？**
A:
- 确保微信版本在7.0以上
- 尝试重启微信
- 检查网络连接
- 清除微信缓存

**Q: 云函数调用失败？**
A:
- 检查网络连接
- 确保已授权微信登录
- 尝试重新打开小程序
- 如持续失败，请联系客服

**Q: 数据不同步？**
A:
- 检查网络连接
- 等待几秒后刷新
- 重新登录
- 如问题持续，请联系客服

### 账户问题

**Q: 如何修改个人信息？**
A:
- 网页版：登录后进入个人中心
- 小程序：点击"我的" → 个人资料

**Q: 可以注销账户吗？**
A:
- 目前版本不支持自助注销
- 请联系客服处理
- 我们会尽快完善此功能

### 支付问题

**Q: 如何充值？**
A:
- 目前使用积分系统
- 通过活动获得积分
- 未来可能支持真实货币充值

**Q: 积分可以兑换吗？**
A:
- 可以在技能市场购买技能
- 可以兑换奖励
- 未来支持更多兑换方式
